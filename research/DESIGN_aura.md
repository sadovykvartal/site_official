# Дизайн-аналіз прототипу AURA (зі скрінів власника + live CSS)

> Дата: 2026-05-29
> Джерело: 8 фулл-скрінів desktop + реальний CSS (`webflow_real.css`)
> Це візуальна специфікація для піксельної копії під ЖК Садовий квартал.

---

## 1. Палітра (точні значення з CSS + звірка зі скрінами)

| Роль | Hex | Де застосовано |
|------|-----|----------------|
| Фон основний | `#FFFFFF` | сторінка |
| Текст-заголовок | `~#11151A` (near-black navy) | великі H2/H3 |
| Текст-body | `~#6E6E6E` сірий | підзаголовки, описи |
| Золото/бронза акцент | `#9D8348` | логотип, активний пункт акордеону, іконки, CTA-кнопка, чек-маркери |
| Беж світлий | `#C3B29D` | вторинний акцент |
| Темно-коричневий | `~#4A3A28` | мовні «бульбашки» маркерів (Будинок 1-5, номери на карті) |
| Картки-беж | `~#EDE6DD` | секція «Високі стандарти» (6 карток) |
| Картки-сірі | `~#F1F1F1` | картки планувань квартир |
| Бордер/лінія | `#E8E8E8` | роздільники акордеону, бордери |
| Тіні | `rgba(0,0,0,.05)`, `rgba(0,0,0,.1)` | картки, кнопки |

> Це палітра ПРОТОТИПУ. Фінальну палітру ЖК Садовий квартал надасть власник - токени винесемо в Tailwind config, щоб міняти в 1 місці.

## 2. Типографіка
- **Заголовки:** жирний геометричний sans-serif, трохи стиснутий, near-black. Кандидати: Manrope / Onest / Inter (з підтримкою кирилиці). Точний шрифт оригіналу - уточнити; для старту беремо **Manrope** (добра кирилиця).
- **Body:** той самий sans, regular, сірий.
- **Логотип «AURA»:** тонкий елегантний display із декоративним підкресленням-розчерком (золото). Для нас - «Садовий квартал» з аналогічним преміум-трактуванням (текст-лого поки що).

## 3. Секції - візуальні деталі (desktop)

### Header (sticky, білий, ~80px)
- Зліва: текст-лого золотом + swash-лінія під ним
- Центр: меню (5 пунктів) near-black, середня вага
- Справа: вертикальний роздільник | телефон (bold) | золота заповнена кнопка «Отримати консультацію» (заокруглення ~6px)

### Hero (повноекранне фото, ~100vh)
- Фонове фото ЖК на весь екран, легке затемнення знизу
- Зліва внизу: pill з пін-іконкою «м. Ужгород, вул. Кримська, 24» (напівпрозорий)
- Великий H1 «AURA» (≈80px, білий) + слоган під ним (білий, ~22px)
- 2 кнопки: золота заповнена «Записатися на перегляд» + текстова «Переглянути планування ↓»
- Справа внизу: 3 frosted-glass картки (Концепція / Архітектура / Розташування) - напівпрозорі, blur, тонкий бордер, заголовок + опис

### Концепція - реалізована як 3 скляні картки в Hero (див. вище)

### Про проєкт (2 колонки)
- Зліва: badge «Про проєкт» (світло-сірий фон, золота ліва смужка) → великий H2 → підзаголовок → **акордеон** (5 пунктів, роздільники #E8E8E8, активний - золотий текст + стрілка →, решта - сіра стрілка ›)
- Справа: велике фото у скруглених кутах (~24px radius); під ним підзаголовок «Тиша, яка наповнює» + абзац + 3 міні-колонки (заголовок золотий + опис сірий)

### Високі стандарти (6 карток, 3×2)
- По центру H2 + підзаголовок
- Картки беж `#EDE6DD`, radius ~16px, padding великий
- Кожна: золота line-іконка (вгорі) → заголовок (bold near-black) → опис сірий

### Галерея «Середовище, в якому хочеться бути»
- По центру H2 + підзаголовок
- Горизонтальна карусель **портретних** зображень (radius ~16px), видно ~4.5 карток, активна по центру крупніша
- Під каруселлю: 2 круглі кнопки ‹ › (білі, тонкий бордер)

### Інфраструктура-екосистема
- По центру H2 «Екосистема...» + підзаголовок
- Велике аерофото (radius ~16px) з накладеними маркерами: коричневі «бульбашки» (Будинок 1-5) + круглі білі бейджі з номерами 1-12

### Локація (карта + список)
- Зліва: золота line-art мапа міста з іконками-пінами + центральна бульбашка-лого
- Справа: H2 (2 рядки) + підзаголовок + сітка 3×4 зручностей: золоте коло з line-іконкою + назва (bold) + дрібний опис

### Квартири
- По центру badge «Квартири» → H2 → підзаголовок
- Селектор «Будинок: 1 2 3 4 5» - активний коричнева заповнена pill, решта білі з бордером; справа ‹ ›
- Картки планувань: сірий фон `#F1F1F1`, radius ~20px, всередині креслення; знизу панель дій (3 іконки: поверхи / кімнати / fullscreen)

### Технічні характеристики
- Зліва: великий H2 (3 рядки) + підзаголовок
- Справа: сітка 3 колонки × 6 рядків; кожен пункт: золоте коло з ✓ → текст (короткий, «Параметр: значення»)

### Далі (з RESEARCH): Ціни, Хід будівництва, Відділ продажу+форма, Забудовник, Footer - скрінів немає, беремо зі структурного ресерчу + live-сайту.

## 4. Загальні принципи стилю
- Багато повітря, широкі відступи між секціями (~120px desktop)
- Контейнер ~1280-1320px, центрований
- Скруглення: картки 16-24px, кнопки/pill 6-8px (CTA) та повні (селектори)
- Іконки: тонкі лінійні (stroke), золото
- Мінімалізм, преміум, спокій; акцент лише золото + темно-коричневий
